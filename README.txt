Este laboratorio consiste en el desarollo de un pagina web.

Se escogio crear una tienda online llamada Techstore Wade donde presentan una variedad de equipos electronicos para la venta.
Esta pagina fue creada en visual studio code utilizando HTML,CSS,Bootstrap y Javascript.

En la pagina principal se presentan un carrusel de imagenes y enlaces a diferentes parte de la pagina web.Bootstrap fue utilizado para crear un diseño responsivo y JavaScript se implementó para agregar interactividad a los productos y validar el formulario de contacto.
El proyecto fue almacenado en Github.
